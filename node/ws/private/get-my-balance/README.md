# Get my balance

Subscribe to my balance.

## Run

1. Install dependencies with `npm install`
2. Run the sample with `npm run start`

## Output

```
{
    "event": "my-balance",
    "data": {
        balance: [
            {
                currency: 'B2M',
                balance: 1000,
                blockedBalance: 1000
            },
            {
                currency: 'EUR',
                balance: 100
                blockedBalance: 200
            }
        ]
    }
}
```
