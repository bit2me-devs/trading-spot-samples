# Get my working-capital

Subscribe to my working-capital.

## Run

1. Install dependencies with `npm install`
2. Run the sample with `npm run start`

## Output

```

        "event":"my-working-capital",
        "data":
        {
            "userId":"2bf436fc-43e6-459a-b647-6b446f72ad96",

                }
```
